# AI 寫作伴侶 - 使用者操作說明

本文檔將引導您如何設定、啟動並使用「AI 寫作伴侶」應用程式。

## 1. 環境設定

在開始之前，請確保您的系統已安裝 Python。

### 第一步：安裝必要的 Python 套件

開啟您的終端機（命令提示字元），進入專案的根目錄，然後執行以下指令來安裝所有必要的 Python 函式庫：

```bash
pip install -r requirements.txt
```

### 第二步：設定 API 金鑰

應用程式需要 Google 和 Hugging Face 的 API 金鑰才能正常運作。

1.  在專案根目錄下找到 `.env` 檔案。如果不存在，請建立一個。
2.  在 `.env` 檔案中，加入以下兩行，並將 `YOUR_API_KEY` 替換成您自己的金鑰：

    ```
    GOOGLE_CLOULD_API_KEY="YOUR_GOOGLE_API_KEY"
    HUGGINGFACEHUB_API_TOKEN="YOUR_HUGGINGFACE_API_TOKEN"
    ```

    - `GOOGLE_CLOULD_API_KEY`：用於「讀者 AI」和「編劇 AI」的核心功能。
    - `HUGGINGFACEHUB_API_TOKEN`：用於建立知識庫時的文本嵌入功能。

## 2. 啟動應用程式

完成環境設定後，在終端機中執行以下指令即可啟動應用程式：

```bash
streamlit run app.py
```

執行成功後，您的預設瀏覽器將會自動開啟一個新分頁，顯示應用程式的操作介面。

## 3. 功能介紹與操作指南

應用程式介面主要分為左右兩欄。

### 左欄：小說編輯與讀者評價

#### 3.1 小說編輯與儲存

此功能讓您可以方便地編輯小說內容並存檔。

1.  **上傳檔案**：點擊「上傳 Word 檔以開始編輯」按鈕，選擇您的 `.docx` 格式小說檔案。
2.  **線上編輯**：檔案內容會自動載入到下方的「在這裡編輯你的小說內容」文本框中。您可以直接在此進行修改、新增或刪除。
3.  **儲存與下載**：編輯完成後，點擊「儲存並下載」按鈕。系統會生成一個名為 `你的小說.docx` 的新檔案，接著點擊「點此下載新文件」即可將檔案儲存到您的電腦。

#### 3.2 讀者 AI 的讀後評價

您可以讓 AI 針對您的某段文稿提供讀後感。

1.  **貼上內容**：將您想讓 AI 閱讀的任何文字片段，複製並貼到「將你想給 AI 閱讀的片段貼在這裡」的文本框中。
2.  **獲取評價**：點擊「請讀者 AI 給出評價」按鈕。
3.  **查看結果**：稍待片刻，AI 的讀後評價將會顯示在按鈕下方。

### 右欄：編劇 AI 聊天室 (RAG)

此功能是應用程式的核心，您可以建立自己的知識庫（例如世界觀、角色設定），並讓 AI 根據這些背景資料與您進行深度對話。

#### 3.3 建立新知識庫

1.  **展開建立區**：點擊「建立新知識庫」以展開設定選項。
2.  **命名知識庫**：在「輸入新知識庫名稱」欄位中，為您的知識庫取一個獨一無二的名稱（例如：`我的魔法世界`）。
3.  **上傳設定檔**：點擊「上傳檔案以建立知識庫」，選擇一個或多個包含設定資料的檔案（支援 `.docx` 和 `.txt` 格式）。
4.  **選擇文件類型**：指明您上傳的檔案主要內容類型，例如「世界觀設定」或「人物設定」。
5.  **開始建立**：點擊「建立知識庫」按鈕。系統會開始處理檔案並建立向量資料庫。建立成功後，這個新的知識庫會出現在上方的選擇列表中，並自動被選取。

#### 3.4 與編劇 AI 聊天

1.  **選擇知識庫**：在「選擇要使用的知識庫」下拉選單中，勾選一個或多個您希望 AI 在對話時參考的知識庫。啟用成功後，下方會顯示「已啟用知識庫：...」的提示。
2.  **開始對話**：在底部的「請輸入你的問題或想法...」輸入框中，輸入您想問的問題或想討論的點子，然後按下 Enter。
3.  **獲取回覆**：AI 會根據您選擇的知識庫內容，生成具有針對性的回覆。例如，您可以問：「幫我根據[角色A]的個性，設計一段他初次見到[角色B]時的場景。」
4.  **清空歷史**：如果您想開始一段全新的對話，可以點擊「清空聊天歷史」按鈕。
